# Valeria Jauregui Lorda - Portfolio

**Live Preview:** [valeryjl.com.ar](https://valeryjl.com.ar)

This repository contains the source code for my personal portfolio website, designed to resemble a classic RobCo Industries terminal from the Fallout universe. 

## Features
- **Authentic Fallout Aesthetic:** Pure HTML/CSS/JS implementation of a VT-style terminal.
- **Dynamic Typing Effect:** Simulates a boot sequence on load.
- **Section Toggling:** JavaScript-powered smooth transitions between content screens without page reloads.
- **CRT CRT Simulation:** Features scanlines, visual distortion, vignetting, and a phosphor glow effect entirely in CSS.

## Projects Showcased
- **AF Projects:** Core systems focused on service management and web interfaces.
- **Experimental:** Various projects including AI chat integration (InsanusChat) and TypeScript tools.
